jQuery

The Document Ready Event

A good idea is to use jQuery after the whole document is loaded. This can be realized by wrapping all the jQuery lines inside the The Document Ready Event by using the syntax:

$(document).ready(function(){ // jQuery methods go here... });

There is also a shortcut syntax for this:

$(function(){ // jQuery methods go here... });

jQuery Selectors

jQuery selectors are used to "find" (or select) HTML elements based on their name, id, classes, types, attributes, values of attributes and much more. It's based on the existing CSS Selectors, and in addition, it has some own custom selectors. All selectors in jQuery start with the dollar sign and parentheses: $().

Since the selector supports CSS format, we can use $("tag") to select tag, $(".class") to select class, $("#ID") to select ID and so on. Following table shows some more examples:

Selector Description
$("*") Selects all elements
$(this) Selects current element
$("p.intro") Selects all p elements with class="intro"
$("p:first") Selects the first p element
$("ul li:first") Selects the first li element of the first ul
$("ul li:first-child") Selects the first li element of every ul
$("[href]")) Selects all elements with an href attribute
$("a[target='_blank']") Selects all elements with a target attribute value equal to "_blank"
$("a[target!='_blank']") Selects all elements with a target attribute value NOT equal to "_blank"

jQuery Event Methods

jQuery is tailor-made to respond to events in an HTML page.

In jQuery, most DOM events have an equivalent jQuery method. To assign a click event to all paragraphs on a page, you can do this:

$("p").click();

The next step is to define what should happen when the event fires. You must pass a function to the event:

$("p").click(function(){ // action goes here!! });

The on() Method

The on() method attaches one or more event handlers for the selected elements. For example:

$("p").on("click", function(){ $(this).hide(); });

We can even attach multiple event handlers to a single element:

$("p").on({ mouseenter: function(){ $(this).css("background-color", "lightgray"); }, mouseleave: function(){ $(this).css("background-color", "lightblue"); }, click: function(){ $(this).css("background-color", "yellow"); } });

jQuery hide() and show()

We can use hide() and show() methods to hide or show a HTML element.

The hide() and show() methods can also take twp optional parameters

  • speed - the speed of the animation in milliseconds
  • callback - A callback function to be invoked after the animation completes

Here is an example $("button").click(function(){ $("p").hide(1000); });

jQuery toggle()

We can also toggle between hiding and showing an element with the toggle() method. Shown elements are hidden and hidden elements are shown: $("button").click(function(){ $("p").toggle(); });

jQuery Fading Methods

With jQuery you can fade an element in and out of visibility. jQuery has the following fade methods:

  • fadeIn() - fades an element in
  • fadeOut() - fades an element out
  • fadeTo() - fades an element to a certain opacity
  • fadeToggle() - toggles between fadeIn and fadeOut

Just like the hide and show methods, these methods also takes speed as an additional parameter.

I'm a fading div

jQuery fadeTo() Method

This allows fading to a given opacity (value between 0 and 1). For example: $("button").click(function(){ $("#div1").fadeTo("slow", 0.15); $("#div2").fadeTo("slow", 0.4); $("#div3").fadeTo("slow", 0.7); });

jQuery - Get Content and Attributes

Three simple, but useful, jQuery methods for DOM manipulation are:

  • html() - gets or sets the HTML content of an element
  • text() - gets or sets the text content of an element
  • val() - gets or sets the value of an input element

For example:
$("#btn1").click(function(){ alert("Text: " + $("#test").text()); }); $("#btn2").click(function(){ alert("HTML: " + $("#test").html()); });

This text will be used in the alert.

The same above three methods can also be used to set the values.

$("#btn1").click(function(){ $("#test1").text("Hello world!"); }); $("#btn2").click(function(){ $("#test2").html("Hello world!"); }); $("#btn3").click(function(){ $("#test3").val("Dolly Duck"); });

Set Attributes - attr()

The jQuery attr() method is also used to set/change attribute values. The following example demonstrates how to change (set) the value of the href attribute in a link:

$("button").click(function(){ $("#w3s").attr("href", "https://www.w3schools.com/jquery/"); });
